## Repository Structure

```
uArm_Metal/
├── Assembly_Resources/
├── CAD_Files/
├── Calibration_Files/
├── Documentation/
├── Useful_Links.txt
└── README.md
```

### Assembly_Resources

Contains all resources related to the assembly of the robotic arm.

* Assembly manual
* Assembly videos

### CAD_Files

Contains the original mechanical design files of the uArm Metal.

* STEP files
* STL files
* Assembly files

These files can be used for visualization, modification, manufacturing, or replacement of mechanical components.

### Calibration_Files

Contains the files required to calibrate the robotic arm.

* Arduino calibration
* Calibration data (Excel)

The calibration code requires the official **UArmForArduino** library before compilation.

### Documentation

Contains technical documentation related to the uArm Metal.

Examples include:

* Technical documentation
* Additional reference documents

### Useful_Links.txt

Contains links to official resources and documentation, including:

* Official uArm Arduino library
* Official documentation
* Additional reference pages

## Requirements

To use the calibration software, install the official **UArmForArduino** library before compiling the Arduino project.

## Purpose

This repository serves as a centralized collection of resources for users who need to assemble, maintain, calibrate, repair, or study the **uArm Metal** robotic arm. It is intended to simplify access to all relevant documentation, software, CAD models, and supporting materials in a single location.
